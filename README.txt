# Horiseon Website
***
A website that contains a header with an internal navigation portion to 3 different elements in the main body or "content" section of the page.
The page also contains a Side Panel to the right of the Main Body "content" section.
The side panel contains only decorative icon type images, headers, and text.
The each of the 3 div elements in the main body "content" section contains a header, image and text. 

Code was updated to include semantic elements and accesibility improvements as well as changes to make the CSS more concise.

link to the deployed page: https://emmccormick.github.io/module-1-chal/
